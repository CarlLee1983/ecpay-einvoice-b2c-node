# Support

Thank you for using ecpay-einvoice-b2c-node! Below are the recommended ways to get help.

## Quick help (recommended)
- For usage questions, examples, and troubleshooting: use GitHub Discussions
  - https://github.com/CarlLee1983/ecpay-einvoice-b2c-node/discussions
  - Choose the category that best fits: Q&A, How-to, Show & Tell, Ideas

## Bug reports
- If you found a bug (unexpected behavior or regression), please open an Issue:
  - https://github.com/CarlLee1983/ecpay-einvoice-b2c-node/issues
- Provide: Node version, SDK version, reproduction steps, minimal code example and logs.

## Security
- Do NOT post security issues publicly.
- See SECURITY.md for private reporting (we accept email or GitHub private vulnerability reports).
